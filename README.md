# React Contact App

![GitHub last commit](https://img.shields.io/github/last-commit/sinansarikaya/react-contact-app?style=flat-square)

<hr />

### Installation Options

```
$ git clone https://github.com/sinansarikaya/react-contact-app.git
```

```
$ npm install
```

```
$ npm start
```
